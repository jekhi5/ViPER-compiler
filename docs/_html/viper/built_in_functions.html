Built-in Functions

There are a handful of functions that make up the ViPER standard library. These are available in all programs. While they may be implemented in C, they behave identically to user-defined functions.

Type Predicates

These are unary functions that test if a value is a given type.

isnum

isnum(obj) : Any -> bool

Checks if obj is a number.

isbool

isbool(obj) : Any -> bool

Checks if obj is a boolean.

istuple

istuple(obj) : Any -> bool

Checks if obj is a tuple.

Arithmetic

add1

add1(n) : number -> number

Computes n + 1.

sub1

sub1(n) : number -> number

Computes n - 1.

IO Operations

print

print(obj) : Any -> Any

Prints obj to stdout. Returns obj.

input

input() : -> number

Reads a numeric value from stdin.

Equality

equal

equal(obj1, obj2) : Any Any -> boolean

Compares two objects for equality. Computes deep equality on tuples.

    let 
        a = (1, (2, 3),
        b = (1, (2, 3),
        c = ((1, 2), 3)
    in
        (equal(a, b), equal(a, c))

    (true, false)
